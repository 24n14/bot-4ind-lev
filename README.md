# 🚀 Торговый бот 4ind-lev

Торговый бот для автоматической торговли на криптовалютных биржах с использованием консенсусного анализа 4 индикаторов и фильтрации по уровням поддержки/сопротивления.

## 📊 Ключевые возможности

### 🎯 Консенсусный анализ индикаторов
Бот использует 4 технических индикатора для принятия решений:
- **MA/EMA пересечения** - определение тренда
- **MACD** - анализ импульса и разворотов
- **Stochastic** - определение перекупленности/перепроданности
- **OBV (On-Balance Volume)** - анализ объёма

### 🛡️ Фильтрация по уровням
- Автоматическое определение уровней поддержки и сопротивления
- Блокировка входов вблизи исторических экстремумов
- Интеллектуальная фильтрация ложных сигналов

### ⚙️ Управление рисками
- Автоматический Stop-Loss и Take-Profit
- Trailing Stop для защиты прибыли
- Контроль размера позиции
- Работа на демо-счёте для тестирования

### 🔄 Автоматическая торговля
- Работа 24/7 без вмешательства пользователя
- Автоматический разворот позиций при сильных сигналах
- Синхронизация с закрытием свечей
- Поддержка прокси для обхода ограничений

## 🎮 Управление ботом через config.py

Бот полностью настраивается через файл `config.py`. Даже новичок сможет легко настроить все параметры!

### 📋 Быстрый старт для новичков

1. **Создайте файл `.env`** в той же папке, где находится `config.py`
2. **Добавьте в него свои ключи**:
   ```
   API_KEY=ваш_api_ключ_сюда
   SECRET_KEY=ваш_секретный_ключ_сюда
   PROXY_USER=логин_прокси
   PROXY_PASS=пароль_прокси
   ```
3. **Настройте основные параметры** в `config.py`

### ⚙️ Основные настройки (самое важное!)

#### 🔐 Настройки подключения
```python
DEMO_TRADING = True  # True для тестирования, False для реальной торговли
SYMBOL = 'BTC/USDT:USDT'  # Торговая пара
AMOUNT = 0.01  # Размер позиции в BTC
LEVERAGE = 10  # Кредитное плечо
```

#### ⚠️ Параметры риска (настройте осторожно!)
```python
STOP_LOSS = 1  # Stop-Loss в процентах
TAKE_PROFIT = 1  # Take-Profit в процентах
TRAILING_STOP_DISTANCE = 150  # Дистанция для Trailing Stop
MIN_AMOUNT = 0.001  # Минимальный размер позиции
```

#### 📈 Параметры анализа
```python
TIMEFRAME = '5m'  # Таймфрейм: 1m, 5m, 15m, 1h, 4h, 1d
MIN_CONFIDENCE = 0.5  # Минимальная уверенность для входа (0.0-1.0)
```

### 🎯 Настройка индикаторов

#### Веса индикаторов (важность каждого)
```python
INDICATOR_WEIGHTS = {
    'ma_crossover': 1.0,  # Вес пересечения MA/EMA
    'macd': 1.0,          # Вес MACD
    'stochastic': 1.0,    # Вес Stochastic
    'obv': 1.0            # Вес OBV тренда
}
```

#### Параметры индикаторов
```python
# MA/EMA
MA = 2    # Период быстрой MA
EMA = 50  # Период медленной EMA

# MACD
FAST_macd = 12   # Быстрая линия
SLOW_macd = 26   # Медленная линия
SIGNAL_macd = 9  # Сигнальная линия

# Stochastic
FASTK = 14  # Период %K
SLOWK = 3   # Сглаживание %K
SLOWD = 3   # Сглаживание %D

# OBV
OBV_SMA_PERIOD = 5   # Сглаживание OBV
EMA_50_PERIOD = 50   # EMA для тренда
ATR_PERIOD = 14      # Период ATR для волатильности
```

### 🔧 Продвинутые настройки

#### Прокси (если нужен)
```python
USE_PROXY = True  # Включить/выключить прокси
PROXY_HOST = '154.219.207.178'  # Адрес прокси-сервера

```

## 📊 Как работает бот

### 🔄 Торговый цикл
1. **Ожидание закрытия свечи** - бот ждёт закрытия текущей свечи
2. **Сбор данных** - загружает исторические данные
3. **Анализ индикаторов** - рассчитывает 4 индикатора
4. **Консенсусный анализ** - определяет общий сигнал
5. **Фильтрация по уровням** - проверяет уровни поддержки/сопротивления
6. **Исполнение сделки** - открывает/закрывает позиции

### 🎯 Типы сигналов
- **Bullish** (бычий) - рекомендация к покупке
- **Bearish** (медвежий) - рекомендация к продаже
- **None** - отсутствие чёткого сигнала

### 🛡️ Защитные механизмы
- **Блокировка у экстремумов** - не позволяет войти у исторических максимумов/минимумов
- **Минимальная уверенность** - фильтрует слабые сигналы
- **Проверка ликвидности** - контролирует минимальный размер лота

## ⚠️ Важные предупреждения

### Для новичков:
1. **Всегда начинайте с `DEMO_TRADING = True`**
2. **Начинайте с маленьких сумм** - `AMOUNT = 0.001` без плеча будет стоить ~78$  
3. **Используйте маленькое плечо** - `LEVERAGE = 10` # это самое маленькое плечо если маленький депозит (100$)
4. **Установите Stop-Loss** - `STOP_LOSS = 0.5 - 1`

### Рекомендации по настройке:
- **Консервативный стиль**: `MIN_CONFIDENCE = 0.7`, `LEVERAGE = 3`
- **Агрессивный стиль**: `MIN_CONFIDENCE = 0.4`, `LEVERAGE = 10`
- **Тестовый режим**: `DEMO_TRADING = True`, `AMOUNT = 0.001`

## 🔧 Устранение неполадок

### Частые проблемы:
1. **"Не удалось подключиться к бирже"**
   - Проверьте API ключи в `.env`
   - Убедитесь, что `DEMO_TRADING` правильно настроен

2. **"Недостаточно свечей"**
   - Увеличьте `LIMIT = 300` в `config.py` 
   - Проверьте подключение к интернету

3. **Бот не открывает позиции**
   - Уменьшите `MIN_CONFIDENCE = 0.4`
   - Проверьте `MIN_CONSENSUS_WEIGHT` и `MIN_PARTICIPATION`

## 📈 Советы по использованию

### Для тестирования:
1. Запустите бота на демо-счёте неделю
2. Анализируйте логи в папке `logs/`
3. Настройте параметры под свою стратегию

### Для реальной торговли:
1. Протестируйте на разных таймфреймах
2. Начните с минимальной суммы
3. Постепенно увеличивайте размер позиции
